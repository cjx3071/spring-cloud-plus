###  **spring-cloud-plus** 
spring-cloud-plus 是以spring-cloud-alibaba为基础并整合一些常用框架的分布式基础开发平台。
这个项目是我自己维护的实践型项目，平时工作中遇到的一些常用框架和技术，我都会整合到此项目，并持续更新。
### 目标
- 成为一个简单易用、快速高效、功能丰富、安全稳定的分布式项目脚手架。
- 让每个人都可以独立、快速、高效地开发一套分布式项目！
